Day 3 : Nested List & Tables in HTML

Topics Covered:

  1. Ordered list (ol)
  2. Unordered list (ul)
  3. Nested list : List inside a list (can be ol or ul)
  4. Use : li * (no. of list u want)



Ordered List

  1. We use <ol> to initiate it.
  2. Gives List in a numbered format

Unordered List

  • We use <ul> to initiate it
  • Gives List in a dotted format

Table : <table>

  • <tr> → table row
  • <td> → table data
Tables are written as: <table border="1">
<tr>
<td></td>
<td></td>
</tr>
<tr>
<td></td>
<td></td>
</tr>
</table>
Professional Way to write table: <table border="1">
<thead>
<tr>
<td></td>
<td></td>
</tr>
</thead>
<tbody>
<td></td>
<td></td>
</tr> </tbody>
<tfoot>
<tr>
<td colspan="2"></td>
<td colspan="3"></td>
<td rowspan="2"></td>
</tr>
</tfoot>
</table>
  1. Attribute given to th or td
  2. colspan → spanning column to desired no. of columns
  3. rowspan → spanning for rows (Given to th but mostly td)
