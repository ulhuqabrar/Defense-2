Day 2: HTML, Tags and Attributes

Topics Covered:

  1. HTML
  2. Headers
  3. Paragraph
  4. List
  5. Links
  6. Attributes



  1. HTML

    • Hyper Text Markup language
    • HyperText = hyperlink (helps to land)
    • Markup Language = Marking to makeup the text
      • <!-- --> → comment
  2. Code should be written in VS Code

    Any file with .html will be considered HTML file

  3. Tags

    Helps make the text beautiful by modifying.

  4. Header tag :- h1 to h6

    • h1 → biggest, h6 → smallest
    • weightage helps in SEO.
    • It is recommended to have only 1 h1 in a website.
    • Headings shouldn’t jump.

  5. Paragraph tag :- <p> </p>

    helps to write paragraph
    <hr/> – draws line
    <br/> – breaks & continues in new line

  6. List tags:

    • <ol></ol> → ordered list. & <li></li> → items in it.
    • <ul></ul> → unordered list.

  7. Link addition:

    <a href> </a>

    • (anchor tag) → reference link (hyper reference).

    Optional way to link:

    <a href="link">Name <a href="link" target="_blank">Name

  8. Attributes

    shows properties of a tag; placed inside the tag.

    href ; target can be named attributes.

  9. Image Tag

    <img src="source" height="" width="" alt="Message when no image">
