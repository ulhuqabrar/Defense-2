  1. File Path

    When Using file(obj) of the same folder, using './' is recommended (used in js).

    They are of 2 types:
    Relative Path Absolute Path
    destination from the current position United Universal path (when we don't know current position)


    For down level in Relative Path For upper level in Relative Path For absolute Path
    folder1/folder2/../file(obj) ../file(obj) Copy file location from file manager/explorer but system blocks so using online images is preferred here.

    Upper level relative & absolute they act as one unit.

    Path blocked by the browser when it try to access the system for security reasons.

  2. Boiler Plate Code

    Written as

    <!DOCTYPE html> -> use HTML5
    <html> -> file
    <head> -> necessary but not to be
    <title> -> Display Name Shown </title>
    </head>
    <body>
    Code Content
    </body>
    </html>

    UTF8 -> 2-4 bytes.
    ASCII -> 1 byte.
